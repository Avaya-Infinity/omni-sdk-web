# Change Log

## v1.0.2 (Dec 12, 2025)

- Added support for Italian language and updated other locales.
- Added new method `AvayaInfinityOmniSdkMessagingUi.getAvailableLocales()` to get the list of all the locales for which translation resources have been loaded in Messaging UI.
- Added fix for CSS bleeding issue.
- Known Issues
  - Print Conversation: Messaging UI fonts do not render correctly when printing.

## v1.0.1 (Nov 18, 2025)

- Multiple bug fixes and translation improvements.

## v1.0.0 (June 24, 2025)

- Introduction of Avaya Infinity Omni SDK modules to support web chat capabilities

  - Omni SDK Core
  - Omni SDK Messaging
  - Omni SDK Messaging UI
